12.6.1 HTML export commands

C-c C-e h h     (org-html-export-to-html)
Export as a HTML file. For an Org file myfile.org, the HTML file will be myfile.html. The file will be overwritten without warning. C-c C-e h o Export as a HTML file and immediately open it with a browser.
C-c C-e h H     (org-html-export-as-html)
Export to a temporary buffer. Do not create a file.
