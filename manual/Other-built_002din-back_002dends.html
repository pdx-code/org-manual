12.11 Other built-in back-ends

On top of the aforemetioned back-ends, Org comes with other built-in ones:

  • ox-man.el: export to a man page.
  • ox-texinfo.el: export to Texinfo format.
  • ox-org.el: export to an Org document.

To activate these export back-end, customize org-export-backends or load them directly with e.g., (require 'ox-texinfo). This will add new keys in the export dispatcher (see The Export Dispatcher).

See the comment section of these files for more information on how to use them.
